================================================================================
                    Dota2 タイマーアプリ まとめ
================================================================================

【1. 概要】
-----------
Dota 2 の主要オブジェクト出現タイミングと昼夜サイクルを管理する Web アプリです。
React + TypeScript + Vite で実装し、サーバーなしで使える単一 HTML にもビルドできます。


【2. 画面構成】
--------------
■ 初期画面
  ・画面中央に「START」ボタンのみ
  ・押下でタイマー画面へ遷移

■ タイマー画面
  ・5つの丸型タイマー（横並び／レスポンシブ）
  ・画面上部：試合時間（mm:ss）、昼夜表示（昼/夜）
  ・タイマー一覧の下：−10秒 / +10秒 ボタン
  ・メモ欄（自由記入）
  ・リセットボタン（押下で初期画面へ戻る）


【3. タイマー一覧（5種類）】
----------------------------
  ID        ラベル              周期
  ----      ------------------  -------
  lotus     ロータス             180秒（3分）
  wisdom    知恵の祠             420秒（7分）
  bounty    バウンティールーン   240秒（4分）
  power     パワールーン         120秒（2分）
  daynight  昼夜                 300秒（5分）


【4. 主な仕様】
--------------
・開始前カウントダウン
  開始ボタン押下後、40秒間「マッチ開始まで ○○s」を表示。
  その間は全タイマーはフル表示のまま。40秒経過後に一斉にカウント開始。

・同時進行
  40秒経過後、5つすべてのタイマーが同時にカウントダウン。

・円形ゲージ
  各タイマーは丸型。上（12時）を起点に時計回りにゲージが減り、
  残り時間に応じて色が緑→黄→赤に変化。

・周期到達時
  0秒になったら自動で次周期へ。その瞬間だけ軽いパルスアニメーション。

・昼夜
  5分ごとに昼⇔夜。背景色が切り替わる。
  昼夜タイマーは ☀️ / 🌙 の絵文字で表示。

・試合時間
  画面上部に「試合時間: mm:ss」を表示（開始前40秒を除いた経過時間）。

・時間調整
  「−10秒」「+10秒」ボタンで経過時間を10秒戻す／進める。

・メモ
  画面下部のテキストエリアにメモを記入可能。
  リセットしても内容は保持。localStorage / sessionStorage に保存し、
  リロード後も復元。

・アイコン
  ロータス・知恵の祠・バウンティー・パワーは public/icons/*.png を参照。
  昼夜は画像ではなく ☀️ / 🌙 の絵文字。


【5. ファイル・フォルダ構成】
-----------------------------
dota_timer/
  ├── public/
  │   └── icons/           タイマー用画像（lotus.png, wisdom.png, bounty.png, power.png）
  ├── src/
  │   ├── App.tsx          状態管理・画面切り替え・メモ永続化
  │   ├── App.css          レイアウト・昼夜・ボタン・メモなどのスタイル
  │   ├── main.tsx
  │   ├── index.css        全体のベーススタイル
  │   └── components/
  │       ├── StartScreen.tsx    開始ボタンのみ
  │       ├── TimerScreen.tsx    タイマー一覧・試合時間・メモ・リセット・±10秒
  │       └── TimerIcon.tsx      1つの丸型タイマー（ゲージ・残り時間・画像 or 絵文字）
  ├── dist/                ビルド出力（単一 HTML で配布可能）
  │   ├── index.html       全 JS/CSS をインライン化した1ファイル
  │   └── icons/           上記 PNG をコピーしたもの
  ├── vite.config.ts       base: './', vite-plugin-singlefile で単一 HTML 化
  └── package.json


【6. コマンド・使い方】
-----------------------
  目的                    コマンド
  ----------------------  ----------------------------------------
  開発                    npm run dev
                           → ブラウザで http://localhost:5173 を開く

  本番ビルド              npm run build
                           → dist/ に出力

  サーバーなしで使う      dist/index.html をダブルクリックで開く（file://）

  配布用にサーバーで配信  npx serve dist
                           → http://localhost:3000 で開く（メモ保存が安定）


【7. 技術メモ】
--------------
・状態
  screenState（start / running）、elapsedSeconds、memo。
  昼夜・各タイマー残りは elapsedSeconds から算出。

・メモ保存
  キー「dota_timer_memo」で localStorage を優先し、
  無ければ sessionStorage から読み、保存時は両方に書き込み。
  file:// ではストレージが使えない環境があるため、
  確実に保存したい場合は「npx serve dist」で http で開く運用を推奨。

・単一 HTML
  vite-plugin-singlefile で JS/CSS を dist/index.html にインライン化。
  アイコン画像は dist/icons/ に別ファイルとして出力され、
  相対パス icons/xxx.png で参照。
